HTML Picture Reference

Picture Tag
Tag Name Description
<picture> Picture The picture HTML element contains zero or more <source> elements and one <img> element to offer alternative versions of an image for different display/device scenarios.
Picture Attributes
Attribute Name Description
media Media The media attribute specifies a media condition (similar to a media query) that the user agent will evaluate for each <source> element.
srcset Sourceset The srcset attribute is used to offer a list of possible images based on size or the display's pixel density.
type Type The type attribute specifies a MIME type for the resource URL(s) in the <source> element's srcset attribute. If the user agent does not support the given type, the <source> element is skipped.
